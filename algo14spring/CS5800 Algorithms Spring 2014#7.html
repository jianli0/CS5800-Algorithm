1. (25 points) A farmer wishes to provide fertilizer for a farm. This fertilizer must have 160 units of Nitrogen, 40 units of Phosphorus, 80 units of Potassium, and 20 units of Sulfur. Environmental regulations require that the farmer use no more than 100 units of Phosphorus, 120 units of Potassium, and 50 units of Sulfur. Three commercial fertilizers are available. They have the following percentages of these nutrients:

FertilizerNitrogenPhosphorusPotassiumSulfurCost per unit
110%10%10%5%$100
220%5%15%10%$150
315%15%5%5%$50

Determine the amount of each type of fertilizer to be purchased so as to minimize the total cost. Also show the amount of each nutrient that is provided by the mix of fertilizers.

To formulate this as an LP problem, use one variable for the quantity of each type of fertilizer. The nutrients are then linear functions of the variabls as follows:

  • total nitrogen is 0.10 * x1 + 0.20 * x2 + 0.15 * x3
  • total phosphorus is 0.10 * x1 + 0.05 * x2 + 0.15 * x3
  • total potassium is 0.10 * x1 + 0.15 * x2 + 0.05 * x3
  • total sulfur is 0.05 * x1 + 0.10 * x2 + 0.05 * x3

The constraints are the following:

  • nitrogen ≥ 160 (or equivalently express using x1, x2 and x3
  • phosphorus ≥ 40
  • phosphorus ≤ 100
  • potassium ≥ 80
  • potassium ≤ 120
  • sulfur ≥ 20
  • sulfur ≤ 50

The objective function is 100 * x1 + 150 * x2 + 50 * x3, and we wish to minimize the objective function.

It is not possible to satisfy the constraints.

2. (30 points) A city has a rectangular grid of streets. A car must travel on a street, so distance between two points is the distance of the shortest path along the streets between the two points. Given a set of points on streets of the city, we wish to find the location that minimizes the total distance to the set of points. Explain how to solve this problem using LP.

To solve this problem, select a coordinate system such that all coordinates are positive or introduce appropriate extra variables. The location is specified using two variables (or four if coordinates can be negative). The total distance to the set of points is a sum of absolute values (two for each point). One must introduce a variable for each absolute value, and each of these variables has two inequalities. One must minimize the total distance subject to the inequalities.

3. (10 points) Solve Question 2 above for the following points: (1,2), (7,0) and (5,6). Prove that your solution satisfies the requirements.

Here is the solution written in the R notation. One can use other notations as long as the notation is specified clearly. 

c <- c(0,0,1,1,1,1,1,1)
b <- c(5,-5,6,-6,1,-1,2,-2,7,-7,0,0)
A <- rbind(
c(1,0,-1,0,0,0,0,0),
c(-1,0,-1,0,0,0,0,0),
c(0,1,0,-1,0,0,0,0),
c(0,-1,0,-1,0,0,0,0),
c(1,0,0,0,-1,0,0,0),
c(-1,0,0,0,-1,0,0,0),
c(0,1,0,0,0,-1,0,0),
c(0,-1,0,0,0,-1,0,0),
c(1,0,0,0,0,0,-1,0),
c(-1,0,0,0,0,0,-1,0),
c(0,1,0,0,0,0,0,-1),
c(0,-1,0,0,0,0,0,-1))
linprog::solveLP(c,b,A)

The best location is (5,2) with total distance 12.

4. (15 points) Minimize the sum of three numbers x, y and z such that

  1. 5x + z is at least 15.
  2. x + 5y is at least 5.
  3. 4x + y is at most 10.
  4. y + 3z is at most -20.
Give a proof that your solution is correct.

Introduce two variables for each of x, y and z. The following are the constraints:

  • 5*(xp-xn) + (zp-zn)≥15
  • (xp-xn) + 5*(yp-yn)≥5
  • 4*(xp-xn) + (yp-yn)≤10
  • (yp-yn) + 3*(zp-zn)≤-20

There is no solution that satisfies the constraints.

5. (20 points) Maximize a + 3b + 2c + 4d - 6e + 5f such that all variables are nonnegative and satisfy the following requirements:

  1. 2a - 6b + 6c + 2d - e + 8f = 11
  2. -4a + 2b + c + 2d + 7e - 2f ≥ 6
  3. -a + 5b - c - 4d + e + 7f = 5
  4. 3a - 4b + 3c + 3d + 3e - f ≤ 9

Express this problem in standard form, and find the solution to the dual problem. Prove your solution is correct using the solution to the dual problem.

Here is the LP problem using R notation: 

c <- c(1,3,2,4,-6,5)
b <- c(11,-11,-6,5,-5,9)
A <- rbind(
c(2,-6,6,2,-1,8),
c(-2,6,-6,-2,1,-8),
c(4,-2,-1,-2,-7,2),
c(-1,5,-1,-4,1,7),
c(1,-5,1,4,-1,-7),
c(3,-4,3,3,3,-1))
linprog::solveLP(c,b,A,maximum=TRUE)

The solution is (0,7.44,0,14.06,0,3.44). The maximum is 95.75.

You must explicitly check that the constraints are all satisfied (3 points).

The dual is (0,3.083,0,5.83,0,11.17) (2 points).

Checking the dual (i.e., proving that the claimed maximum point is correct)

In the following, the first linear function is the objective function, and the second bound is obtained by multiplying the bounds by the dual.
a + 3b + 2c + 4d - 6e + 5f ≤ 21.5a + 3b + 9.18c + 4d + 42.42e + 5f ≤ 95.75
It follows that the objective function is bounded by 95.75. Since the maximum point takes this value, 95.75 has been proven to be the maximum value of the objective function.
