CS5800 Algorithms Spring 2014 Assignment #1

1. (20 points) Write pseudocode for the algorithm to compute the maximum of an array of numbers. Prove that your algorithm is correct.

2. (20 points) Arrange the following functions in decreasing order of growth, and prove that the functions are in this order by using asymptotic notation:

  • log(n*n)
  • n10 + 1.1n
  • 100 + 5 n2 + n4
  • n2 / log(n)
  • log(n) + n1/3
  • log(log(n))
  • log(3n)
  • n!2
  • n-1/3
  • 2n2

3. (20 points) Solve the following recurrences:

  1. T(n) = 3 T(n/9) + Θ(√n)
  2. T(n) = 6 T(n/5) + Θ(n1.1)
  3. T(n) = 3 T(3n/4) + Θ(n4)
  4. T(n) = 2 T(3n/5) + T(n/2) + T(n/3) + Θ(n2 log(n))
  5. T(n) = 3 T(2n/5) + T(n/3) + T(n/5) + Θ(n√n)

4. (20 points) For the following three functions of a positive integer, arrange them by increasing order of growth, and find the crossover points of each pair of functions.

  • 10 log2(n)
  • n2 + n + 3
  • 2n

5. (20 points) Run the randomized order statistic algorithm to find the 10th smallest number of the following array, showing each step and the depth of recursion.

{ 19, 11, 13, 2, 28, 9, 46, 3, 27, 19, 31, 2, 21, 1, 1, 25, 47, 17, 11, 24 }

For the random selections use the following random numbers: { 23, 29, 84, 15, 58, 19, 81, 17, 48, 15, 36, 49, 91, 26, 89, 22, 63, 57, 33 }. If you need a number in a range from a to b (inclusive), use modular arithmetic to select the number in the range. For example, if you need to randomly select an integer in the range from 10 to 15 (inclusive), and your random number is 38, you should compute 38 % 6 = 2, and then select the integer 12.
