1. (20 points) A university department teaches various courses and has faculty to teach the courses. Each faculty member can teach some of the courses, and each faculty member has a maximum number of sections that the faculty member can teach. Each course has a desired number of sections that should be taught, but in practice the desired number is the upper limit on the number of sections of a course. Express this problem as a maximum flow problem.

Construct a DAG with one node as the source, one node as the target, and nodes for the faculty and courses. There are edges from the source to each course, from the faculty to the target, and from each course to each faculty member who can teach the course. Note that the edges could all be reversed. The weight on the edge from a faculty member to the target is the maximum number of sections that the faculty member can teach. The weight on the edge from the source to a course is the desired number of sections of the course. Other edges have infinite (unlimited) weight. A maximum flow from the source to the target gives the maximum number of sections that can be taught subject to the constraints.
